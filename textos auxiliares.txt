Ajuda ¬ Primeiro adicione ao menos uma fita nos campos de formulário e pressione o {{OK}}. ¬ Fitas secundárias ¬ Para embaralhar a segunda fita pressione {{SHUFFLE}}. ¬ Para obter o complemento da primeira fita, pressione {{CC}}. ¬ Movendo fitas ¬ Para mover a fita inferior uma casa a esquerda, pressione {{<< Left}}. ¬ Para mover a fitita inferior uma casa a direita, pressione {{Right >>}}. ¬ Para ver o melhor caso, pressione {{MC}}. ¬ Para voltar a condição inicial, pressione {{R}} ¬ Para limpar o terminal, pressione {{cls}}. ¬ Para ver os créditos, pressione {{ɔɔ}}. ¬
